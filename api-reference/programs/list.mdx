---
title: "List Loyalty Programs"
api: "GET /api/loyalty/v1/programs/"
description: "Retrieve all active loyalty programs with their configuration details and benefits"
---

## Overview

This endpoint returns a list of all available loyalty programs for the company. Each program includes configuration details, tier structure, and available benefits. This information is essential for customer enrollment and program selection.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication
  ```
  Authorization: Bearer {your_api_key}
  ```
</ParamField>

## Query Parameters

<ParamField query="active_only" type="boolean" default="true">
  Filter to show only active programs
  ```
  true
  ```
</ParamField>

<ParamField query="include_tiers" type="boolean" default="false">
  Include detailed tier information for each program
  ```
  false
  ```
</ParamField>

## Response

<ResponseField name="count" type="integer">
  Total number of loyalty programs
</ResponseField>

<ResponseField name="results" type="array">
  Array of loyalty program objects
  
  <Expandable title="Program object">
    <ResponseField name="id" type="string">
      Unique program identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Program display name
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Program description and benefits overview
    </ResponseField>
    
    <ResponseField name="currency_name" type="string">
      Points currency name (e.g., "ZupCoins")
    </ResponseField>
    
    <ResponseField name="currency_symbol" type="string">
      Points currency symbol
    </ResponseField>
    
    <ResponseField name="point_value" type="number">
      Monetary value per point in local currency
    </ResponseField>
    
    <ResponseField name="earning_rate" type="object">
      Points earning configuration
      
      <Expandable title="earning_rate object">
        <ResponseField name="base_rate" type="number">
          Base points earned per currency unit spent
        </ResponseField>
        
        <ResponseField name="bonus_multiplier" type="number">
          Bonus multiplier for special promotions
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="tiers" type="array">
      Program tier levels (if include_tiers=true)
      
      <Expandable title="Tier object">
        <ResponseField name="name" type="string">
          Tier name (e.g., "Bronze", "Silver", "Gold")
        </ResponseField>
        
        <ResponseField name="minimum_points" type="integer">
          Minimum points required to reach this tier
        </ResponseField>
        
        <ResponseField name="benefits" type="array">
          List of benefits available at this tier
        </ResponseField>
        
        <ResponseField name="multiplier" type="number">
          Points earning multiplier for this tier
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="active" type="boolean">
      Whether the program is currently active
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Program creation timestamp (ISO 8601)
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      Last program update timestamp (ISO 8601)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://webmaster.zupers.com.br/api/loyalty/v1/programs/?include_tiers=true" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://webmaster.zupers.com.br/api/loyalty/v1/programs/?include_tiers=true', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const programs = await response.json();
console.log('Loyalty programs:', programs.results);
```

```python Python
import requests

url = "https://webmaster.zupers.com.br/api/loyalty/v1/programs/"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
params = {"include_tiers": True}

response = requests.get(url, headers=headers, params=params)
programs = response.json()
print("Available programs:", programs["count"])
```

## Use Cases

- **Program Selection**: Allow customers to choose their preferred loyalty program
- **Feature Discovery**: Display available benefits and tier progression
- **Marketing Integration**: Use program information for promotional campaigns
- **Mobile App Setup**: Initialize loyalty features with program configuration
- **Admin Dashboard**: Display program overview and configuration options

<Note>
  Programs are typically cached client-side since they change infrequently. Consider implementing cache invalidation when program configurations are updated.
</Note>
